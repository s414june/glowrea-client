# Feature: 群組列表（Lists）

## Goal

讓已登入使用者能管理自己建立的群組（List），並查看特定群組的時間軸貼文。群組是 Mastodon 的私人分組功能，可將追蹤的帳號歸類，方便聚焦閱讀。

---

## Scope

### In Scope

- 顯示使用者建立的群組列表（`GET /api/v1/lists`）
- 點入群組後顯示該群組的時間軸（`GET /api/v1/timelines/list/:id`）
- Loading / empty / error state
- 群組時間軸支援 infinite scroll
- 需登入才可存取，未登入導向 `/login`

### Out of Scope

- 建立 / 編輯 / 刪除群組
- 管理群組成員（新增 / 移除帳號）
- 群組時間軸的即時更新（WebSocket）

---

## User Story

作為已登入使用者，我希望能查看我建立的群組，並閱讀特定群組的貼文，以便專注在特定話題或社群。

---

## User Flow

1. 使用者從「更多」選單進入 `/lists`
2. 頁面顯示群組清單（群組名稱列表）
3. 清單為空時顯示 empty state
4. 使用者點擊某個群組 → 進入 `/lists/:id`
5. 頁面顯示該群組的時間軸貼文（依時間由新到舊）
6. 向下捲動到底部時自動載入更多
7. API 發生錯誤時顯示 error state 與 retry 按鈕

---

## Routes

### 群組列表頁

- **Path**：`/lists`
- **Route Guard**：需登入

### 群組時間軸頁

- **Path**：`/lists/:id`
- **Route Guard**：需登入
- **Route Params**：`id`（群組 ID）

## UI Requirements

### 群組列表頁（`/lists`）

```
┌──────────────────────────┐
│ 群組                      │  ← 頁標題
├──────────────────────────┤
│ 群組名稱 A             >  │
│ 群組名稱 B             >  │
│ ...                       │
└──────────────────────────┘
```

- 每個群組為一個可點擊的列表項，右側顯示 `>` 箭頭
- Empty state：「尚未建立任何群組」

### 群組時間軸頁（`/lists/:id`）

- 頁面標題顯示群組名稱
- 貼文列表：複用 `StatusItem` 元件
- Loading：複用 `LoadingSkeleton`
- Infinite scroll sentinel（`IntersectionObserver`）

## 資料流

### API Endpoints

#### 取得群組列表

`GET /api/lists`

對應 Mastodon API：
```
GET /api/v1/lists
```

回傳格式：
```ts
type ListItem = {
  id: string
  title: string
}

type ListsResponse = {
  items: ListItem[]
}
```

#### 取得群組時間軸

`GET /api/lists/:id/timeline`

query params：
- `maxId?: string`

對應 Mastodon API：
```
GET /api/v1/timelines/list/:id?limit=20[&max_id={maxId}]
```

回傳格式：複用 `TimelinePageResponse`

## Server Route 規格

### `GET /api/lists`

- 呼叫 `requireRequestCredentials` 取得 token
- 呼叫 `GET /api/v1/lists`
- 回傳 `{ items: ListItem[] }`

### `GET /api/lists/:id/timeline`

- 呼叫 `requireRequestCredentials` 取得 token
- 從 path 取得 `id`，query 取得 `maxId`
- 呼叫 `GET /api/v1/timelines/list/:id?limit=20[&max_id=...]`
- 解析 Link header 取得 `nextMaxId`
- 回傳 `TimelinePageResponse`

---

## 錯誤處理

| 情境 | 行為 |
|------|------|
| Mastodon API 回傳非 2xx | 回傳對應 HTTP 狀態碼給前端 |
| 未登入 | 401，middleware 導向 `/login` |
| 群組不存在（404） | 顯示「找不到此群組」error state |
| 列表為空 | 顯示 empty state，不視為錯誤 |
